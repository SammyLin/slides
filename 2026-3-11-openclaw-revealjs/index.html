痛點二是會忘記。解法是成長系統，三個機制。 心跳：HEARTBEAT.md，每小時自動醒來巡檢，有事通知我，我不用催。 而且心跳裡面還包含自我檢視：週日週回顧、雙週自我進化，會自動更新 HEARTBEAT.md。 記憶：MEMORY.md 加 memory/ 每日日記，不需要向量資料庫，Markdown 就夠。 還會同步到 Obsidian，手機上也看得到。 流程：AGENTS.md，做任何事之前先建 Linear 卡，不靠記憶靠制度。 這些都是 Markdown 檔案，大家回去可以直接參考。 【目標時間：75秒】

龍蝦養肥了，要做健康檢查

監控用量 + 主機狀況

自幹 openclaw-exporter 使用 Grafana + Prometheus 監控

痛點三是花太多。解法是監控加模型混搭。 我是 DevOps 出身，自己寫了 Prometheus Exporter 接 Grafana。 日常對話用 MiniMax 便宜模型，一個月 10 美金。 關鍵任務用 Claude Max 訂閱制 — $100/月。 千萬不要全部用最貴的 API 按 token 算，會被嚇死。 【目標時間：60秒】

每個 side project 都走同一套流程。 第一步：我用語音在 Telegram 說一句話，描述我想做什麼。 第二步：Agent 會提出 2-3 個技術方案讓我選。 第三步：選好之後 Agent 自己開發、自己測試，全程用 Linear 追蹤。 第四步：CI/CD 自動部署，我確認沒問題就上線。 比如去背工具，我就說了一句話，1小時後就上線了。 【目標時間：60秒】
